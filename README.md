# BlockCraft
一款基于 C++ 开发的 Windows 控制台沙盒游戏

## 🎮 基础操作
| 按键 | 功能 |
|------|------|
| W/A/S/D | 角色上下左右移动 |
| i/j/k/l | 挖掘方块（对应不同方向/位置） |
| I/J/K/L | 放置方块（对应不同方向/位置） |
| q | 丢弃手中的方块 |
| e | 打开背包 |

## 🎒 背包指令
| 指令/按键 | 功能 | 示例 |
|-----------|------|------|
| e | 关闭背包 | - |
| `/swap num1 num2` | 交换背包中编号为 num1 和 num2 的物品 | `/swap 1 4`（交换第1格和第4格物品） |
